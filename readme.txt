Please run the main.py file to run the program.

Syntax for entering commands in the shell of the canvas:
1. Always assign some straight or branched chain molecule that contains carbon atoms only using "variableName->smiles" syntax. 
2. Use the top slider to rotate about y-axis, the left slider to rotate about x-axis, and the bottom slider to zoom.
3. Select molecule from the molOptions window that is located below the molView window.
